    About this site

    The site is built to be free, legal, and reproducible from the beginning. The match results come from the CC0 international results dataset and the public-domain openfootball World Cup data (scores can arrive minutes or hours after full-time, so this is not a live feed). The team ratings are Elo ratings replayed from scratch over 49,000+ matches since 1872.

    The main point: WC26 Pulse is a case study in agentic-ds Skill: a workflow for making data projects with agents trustworthy. Agents can move quickly, but speed is not evidence. The skill makes the work leave a trail: scripts, checks, findings, decisions, and results that can be audited.

    What that looks like here: A fancier model tried to beat the simple baseline. The test was pre-registered, run on the 2014, 2018, and 2022 World Cups, and the fancier Poisson model lost. Then it lost again after the one allowed retry. So the loss stays visible, the raw-Elo baseline gets the headline, and the Poisson layer stays only as experimental scoreline context. The math, the decisions, and the failure are here: methodology.

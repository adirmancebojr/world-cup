Model card

How the predictions work, exactly

An impressive model is useful only if it can stay honest. The headline odds come from a simple Elo baseline because it earned the right to be there. The fancier scoreline model is still shown, but it is marked as experimental, and both pieces are graded against the games after they are played.

At a glance

Headline model Raw Elo simple baseline won the test
Simulations 20,000 seeded Monte Carlo runs
Scorelines Poisson published as experimental
Tracking Final scores graded after public results land

Principles

Pre-register the test

The metric, baselines, filters, and pass/fail rules were written down before fitting. That keeps the target from moving after seeing the shot.

Keep failures visible

The Poisson model failed the headline test. So it does not get the headline. It remains here only as context, with the word experimental beside it.

Trace every number

Every number should have a path back to a script and an input. If a prediction changes, the old one should not disappear into memory.

Pipeline

  1. Replay Elo from scratch. Every team starts at 1500. Then the script replays roughly 49,000 men's internationals from 1872 onward.
  2. Publish raw-Elo match odds. The draw rate comes from recent competitive internationals. The rest is split by the Elo win expectancy.
  3. Generate experimental scorelines. A Poisson goal model estimates likely scores. It is useful, but it did not earn trust as the main model.
  4. Simulate the tournament. 20,000 futures run through the 2026 format: groups, third-place slots, knockouts, and Elo updates inside each simulated path.
  5. Grade finished matches. After a match ends, it is graded against what the site believed before kickoff: outcome for Elo, exact score for Poisson.

Why raw Elo leads

The pre-registered test used all 192 matches from the 2014, 2018, and 2022 World Cups. Each tournament was predicted in order. No future result was allowed to whisper into the past.

Known limits

There are things this model cannot see. It does not know injuries, suspensions, tactics, or the quiet panic inside a locker room. Public results can lag the real match, so this site does not claim real-time coverage. Historical knockout scores follow the source dataset's extra-time convention, and lower group tie-break rules are approximated when needed.
